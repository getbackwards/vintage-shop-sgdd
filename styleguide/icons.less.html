Iconography

  • stylesheet
icons.less  

Uses a custom icon set generated using https://icomoon.io/.

To add a new icon:

  • Upload selection.json located in src/base/icons to a new project on IcoMoon
  • Add the new icon to the library
  • Generate the Font
  • Under "Preferences" make sure the setting Encode and Embed Font in CSS is selected.
  • Update generated json file
  • Update this file with new icons and the embeded font
  • Commit changes

Glyphicons

Other icons needed that are not found above can be used from the gryphicons icon set.
